Hi team,

I've built the framework for the app in Java 21 with JavaFX and SQLite.
It has a sidebar navigation with the following screens, all with UI layouts in place:

  - Dashboard     : Overview panel showing sleep, screen time, productivity and habit stats
                    with a mini sleep chart and greeting.
  - Sleep Tracker : Log sleep hours and quality with a bar chart showing the past 7 days.
  - Screen Time   : Log time by category (Study/Work, Social Media, etc.) with a pie chart.
  - Habits        : Checklist of daily habits with streak tracking.
  - Goals         : Progress bars for personal goals (sleep, steps, screen time, etc.).
  - Pomodoro      : Timer UI (25-minute focus sessions with session dot tracking).
  - Calendar      : Calendar view for scheduling/reviewing entries.

I've included a pre-built installer for now so you can see what the UI looks like: Tracker.dmg

To install:
  1. Open Tracker.dmg
  2. Drag Tracker to your Applications folder
  3. Run the app

This DMG will only work on macOS

To view on Windows you'll need to run it from source.

Steps:

  1. Install Java 21 (JDK): https://adoptium.net
  2. Install Maven:          https://maven.apache.org/install.html
  3. Clone this repo and open a terminal in the project folder
  4. Run: mvn javafx:run
